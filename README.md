| 插件 | 功能 |
|---|---|
| [ComfyUI-Manager](https://github.com/ltdrdata/ComfyUI-Manager) | 增强 ComfyUI 可用性，提供了安装、删除、禁用和启用 ComfyUI 各种自定义节点的管理功能 |
